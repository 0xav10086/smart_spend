# 💰 SmartSpend - 现代化个人财务看板

**SmartSpend** 是一款基于 Vue 3 和 Node.js 开发的个人财务管理与可视化系统。它不仅是一个记账工具，更是你的“数字财务管家”。通过极具现代感的 UI 和动态交互，帮助你直观掌握每一分钱的去向。

---

## ✨ 核心特性

### 1. 响应式视觉预警

* **动态阈值逻辑：** 当月消费支出超过预算的 **80%** 时，界面背景将通过 `Anime.js` 实现丝滑的色彩过渡（从清爽蓝变为警示红）。
* **多渠道通知：** 实时触发 **Telegram Bot** 消息推送，确保你在超支的第一时间收到提醒。

### 2. 现代化交互体验

* **灵动 UI：** 结合 `Tailwind CSS` 的原子化布局与 `Element Plus` 的组件化能力，打造极简、高端的视觉风格。
* **极致动画：** 利用 `Anime.js` 赋予数据生命力，支持数字滚动、列表飞入以及精美的微交互动画。
* **可视化图表：** 基于 `ECharts` 的深度定制，提供消费趋势、分类构成、收支对比等多维度分析。

### 3. 安全与便捷

* **私有化部署：** 专为 VPS 部署设计，数据存储在本地 `SQLite` 数据库，完全掌控个人隐私。
* **权限控制：** 完善的登录校验机制，确保只有你本人能够访问财务数据。
* **便捷录入：** 支持快速记账表单，并预留了 Telegram Bot 远程记账接口。

## 🛠 技术栈架构

### 前端 (Frontend)

| 技术 | 说明 |
| --- | --- |
| **Vue 3 (Composition API)** | 核心框架，利用组合式 API 实现逻辑解耦。 |
| **Vite** | 下一代构建工具，提供极速的开发响应。 |
| **TypeScript** | 强类型支持，提升代码健壮性与开发体验。 |
| **Pinia** | 官方推荐的状态管理，处理用户信息与财务状态。 |
| **Tailwind CSS** | 原子化 CSS 框架，负责响应式设计与现代感 UI。 |
| **Element Plus** | UI 组件库，用于处理表格、表单等重型组件。 |
| **Anime.js** | 动画引擎，负责背景过渡、数字滚动等高级动效。 |
| **ECharts** | 数据可视化，呈现复杂的财务统计图表。 |

### 后端 (Backend)

| 技术 | 说明 |
| --- | --- |
| **Node.js (Express/NestJS)** | 服务端运行环境，提供 RESTful API。 |
| **SQLite** | 轻量级关系型数据库，无需配置即可运行，方便迁移备份。 |
| **Telegram Bot API** | 用于实现即时通知与远程指令记账。 |

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/0xav10086/smart-spend.git

# 安装前端依赖
cd frontend
npm install

# 启动开发服务器
npm run dev

# 安装后端依赖
cd ../backend
npm install

# 启动后端服务
npm run start

```
